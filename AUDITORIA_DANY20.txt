- UX/UI
- performance
- SEO técnico
- acessibilidade
- responsividade mobile
- conversão

Analise TODO o projeto, sem pular arquivos importantes. Quero uma auditoria “gran finale super master power”, com visão técnica e estratégica.

Objetivo:
1. entender exatamente como o site está hoje
2. identificar tudo o que está bom, ruim, incompleto ou mal resolvido
3. descobrir o que ainda falta para o site ficar nota 1000
4. gerar um plano claro de melhoria com prioridades

Quero que você analise pelo menos estes pontos:

1) Estrutura e arquitetura
- organização de pastas
- componentes repetidos
- acoplamento excessivo
- escalabilidade
- legibilidade do código
- possíveis gargalos de manutenção

2) Front-end e experiência
- consistência visual
- hierarquia da informação
- clareza das seções
- excesso ou falta de conteúdo
- usabilidade
- fluxo de navegação
- qualidade do mobile
- responsividade real
- CTAs fracos ou confusos

3) Performance
- imagens pesadas
- componentes desnecessariamente pesados
- renderização ruim
- carregamento lento
- oportunidades de lazy loading
- otimização de assets
- bundle size
- uso incorreto de fontes, animações, scripts ou bibliotecas

4) SEO
- titles e meta descriptions
- headings (H1, H2, H3...)
- estrutura semântica
- sitemap
- robots.txt
- canonical
- Open Graph
- schema markup
- URLs
- conteúdo raso ou mal otimizado
- páginas sem foco claro de palavra-chave
- oportunidades de melhorar rankeamento

5) Acessibilidade
- contraste
- navegação por teclado
- alt text
- labels
- aria
- semântica HTML
- foco visível
- problemas de leitura para leitores de tela

6) Segurança e boas práticas
- exposição indevida
- uso inseguro de variáveis
- dados sensíveis no front
- dependências suspeitas ou desatualizadas
- headers de segurança recomendados
- riscos básicos

7) Conversão e posicionamento
- se está claro o que o site vende/oferece
- se a proposta de valor está forte
- se gera confiança
- se tem prova social
- se falta autoridade
- se os textos estão convincentes
- se os botões realmente levam o usuário à ação

8) Conteúdo
- clareza
- objetividade
- excesso de texto
- falta de texto
- tom de voz
- coerência entre seções
- erros de escrita
- oportunidades de melhorar comunicação

Formato da resposta:
Quero a resposta em 6 blocos:

BLOCO 1 — Diagnóstico executivo

BLOCO 4 — O que falta para ficar nota 1000

BLOCO 5 — Plano de ação priorizado
- quick wins
- melhorias de médio esforço
- melhorias estruturais
- ordem ideal de execução

BLOCO 6 — Sugestões práticas
- mostre exatamente o que eu deveria ajustar
